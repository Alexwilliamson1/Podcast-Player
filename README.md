# Podcast Player

## Description:

Podcast Player uses the command line to create an application menu and show content for a small collection of contemporary podcast episodes about technology.  Users can display and search for podcast episodes, create a playlist, and show the content of episodes in the playlist.  

The program uses a Model-View-Controller design pattern to manage the processes between the user interface and the program data.  It also uses object-oriented programming techniques, such as abstraction, encapsulation, inheritance, and polymorphism to create arrays of objects from a template class; protect private data members; and allow derived classes to inherit functionality from a base class and override its virtual function. 

## Requirements:

To run this program, one requires a C++17 compiler, GNU Make, and the files in the Media and Videos folders provided.

## Instructions for building and running the program:

To build the program, open a command-line interface, navigate to the directory containing the program’s files, and run the command `make’.  Then, run ‘./PodcastPlayer` to run the program.

## Notes:

This program was tested on macOS 15.7 using Clang.
